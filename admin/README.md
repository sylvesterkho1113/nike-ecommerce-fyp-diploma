# 🛠️ Administrative Control Panel

This directory contains the backend modules for managing the Nike E-Commerce system.

## 📋 Features
- **Dashboard**: Real-time sales summary and recent transactions.
- **Product Management**: Add, edit, delete, and manage inventory status.
- **Category Management**: Organize products into various shoe categories.
- **Order Management**: Track and update customer order statuses.
- **Reporting**: Generate sales reports and export data to Excel.
- **Admin Auth**: Secure login and profile management for staff.

## 📁 Key Modules
- `index.php`: Main admin dashboard.
- `Report.php`: Detailed sales analytics.
- `add_product.php` & `edit_product.php`: Inventory control.
- `order.php`: Customer order fulfillment.
- `export_orders_to_excel.php`: Data portability.
